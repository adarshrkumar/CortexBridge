CortexBridge

Cloud-synced agent instructions delivered via MCP.
One file committed. Every agent stays in sync.

CortexBridge is an MCP server that fetches your project's coding instructions from the cloud and injects them into your AI agent's context automatically — so every agent, on every machine, follows the same rules.

Quick start

Add .cortexconfig to your repo:

project: your-project-id

Then add the MCP server:

claude mcp add cortexbridge --transport http https://mcp.cortexbridge.adarshrkumar.dev/mcp
