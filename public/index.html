CortexBridge

Cloud-synced agent instructions delivered via MCP server.
One file committed. Every agent stays in sync.

Setup

Add a .cortexconfig to your repo:

# .cortexconfig
project: your-project-id

How it works

  • The MCP server reads .cortexconfig on connect
  • Authenticates via OAuth 2.1 (Better Auth)
  • Fetches and returns your instructions as agent context

Why

  • Same instructions across Claude, Cursor, Copilot, Gemini
  • Update instructions in the web UI — no repo PRs
  • No tokens or secrets stored in your repo
